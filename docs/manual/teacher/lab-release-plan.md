## 发布前检查清单

每次发布新实验前确认：

- [ ] 实验卡片（`labs/labN-*.md`）已更新到最新版本
- [ ] 对应的 Book 章节已发布
- [ ] 阶段门禁在 Portal 中已配置
- [ ] 公开测试套件已部署
- [ ] 隐藏测试套件已就绪（不公开）
- [ ] 模板仓库中的 `spec/` 骨架已更新
- [ ] FAQ 已更新常见问题
- [ ] 助教已预演实验流程

## 阶段间的依赖处理

- 阶段 N 的公开投影在阶段 N-1 通过门禁后才对学生可见。
- 模板仓库在学期开始时就包含全部阶段的 spec/ 骨架（但标记为不可见）。
- 学生可以提前阅读后续阶段的 Book 章节（全部公开），但 Lab 卡片和测试在解锁后才可见。
